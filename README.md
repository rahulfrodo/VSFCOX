# VSFCOX
This repository contains the software implementation of the VSFCOX method in
"Variable Selection in Functional Linear Cox Model" by Yue, Self, Wu, Zhang, and Ghosal (2026). 
Download the attached FCOX_selection_Software.zip file containing the source codes and R Markdown illustration (FCOX_selection.html) of the proposed method. 
The readme file within the package provides detailed documentation of the components.
